# Browser & Device Support

## Browser Support Matrix

| Browser       | Desktop | Mobile | Support |
|---------------|---------|--------|---------|
| Chrome        | ✅      | ✅     | Latest 2 versions |
| Firefox       | ✅      | ✅     | Latest 2 versions |
| Safari        | ✅      | ✅     | Latest 2 versions |
| Edge          | ✅      | -      | Latest 2 versions |
| IE 11         | ❌      | -      | Not supported |

## Device Support

### Phones
- iPhone 12+ (iOS 14+)
- Samsung Galaxy S20+ (Android 10+)
- Google Pixel 5+ (Android 11+)

### Tablets
- iPad Air (iOS 14+)
- iPad Pro (iOS 14+)
- Android tablets (Android 10+)

### Desktop
- Minimum 1024px width
- Touch support optional

## Feature Compatibility

| Feature              | Chrome | Firefox | Safari | Edge |
|----------------------|--------|---------|--------|------|
| GSAP Animations      | ✅     | ✅      | ✅     | ✅   |
| ScrollTrigger        | ✅     | ✅      | ✅     | ✅   |
| CSS Grid             | ✅     | ✅      | ✅     | ✅   |
| CSS Flex             | ✅     | ✅      | ✅     | ✅   |
| Swiper Touch         | ✅     | ✅      | ✅     | ✅   |

## Fallbacks

- No JavaScript: Static content still visible
- Low-end devices: Animations disabled via `prefers-reduced-motion`
- Old browsers: Graceful degradation
